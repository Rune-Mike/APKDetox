# APK-Detox
APK智能处理分发系统用于对Android应用进行多重打包，重构APK内部结构、具备包名随机化、签名随机化、APK伪加密、代码混淆、网站封装及加固等功能的自动化分发系统，适用于APK报毒免杀，应用分发等场景，此版为遥辉旗舰版的基础上二次开发重制而来。
apk报毒处理、apk去毒

TG：@Rune_Mike


★ 优化内容

1. 修复一些已知BUG，提升处理效率

2. 升级加固壳支持更高的安卓版本，修改加固密码为动态，修复加固对某些包不能加固的问题

3. 全新现代化UI界面，更加美观提高用户使用体验

4. xml规则已按实际权限名称命名，方便按需选择

5. apk备注功能支持在包管理直接点击更改

★ 新增内容

1. 新增短链接配置功能，可以设置短链接过期时间等

2. 新增XML清单配置文件混淆功能

3. 新增APK伪加密功能

4. DEX文件随机混淆功能

5. 新增OSS对象储存
   · 360智汇、缤纷云、layerpanel

6. 应用包管理界面新增批量删除包功能按钮

7. 更多操作新增下载原包功能按钮
